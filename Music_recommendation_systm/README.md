## Overview
This project implements a song recommendation system based on lyric similarity. It uses TF-IDF (Term Frequency-Inverse Document Frequency) to vectorize song lyrics and cosine similarity to find songs with similar lyrical content.

## Features
- Data collection from Kaggle (Spotify Million Song Dataset).
- Data preprocessing including text cleaning, lowercasing, tokenization, and stop word removal.
- TF-IDF vectorization of song lyrics.
- Cosine similarity calculation between song lyrics to find similar songs.
- A function to recommend songs based on a given song title.
- A Streamlit application for an interactive recommendation interface.

## Dataset
The system utilizes the **Spotify Million Song Dataset** sourced from Kaggle. Due to the large size of the original dataset, a sample of 20,000 songs is used for this application.
